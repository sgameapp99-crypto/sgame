## 安裝與使用

### 環境需求
- Node.js (推薦最新LTS版本)
- Git

### 安裝步驟
1. 克隆專案
```bash
git clone <repository-url>
cd sgame
```

2. 安裝依賴
```bash
npm install
```

3. 啟動開發伺服器
```bash
npm start
```
